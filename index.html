Sales Analysis — Excel

Problem: Identify sales drivers and seasonal dips for a retailer.

Dataset: Transactional sales CSV including date, SKU, region, and revenue.

Tools: Excel (pivot tables, formulas), data cleaning techniques.

Approach: Cleaned data, created pivot tables to summarize revenue by category and time. Built supporting charts to reveal trends and seasonality.

Outcome: Identified top categories responsible for majority of revenue and recommended Q4 inventory adjustments that could reduce stockouts during peak season.

Customer Segmentation — SQL & Power BI

Problem: Prioritize marketing efforts toward highest lifetime value customers.

Dataset: Customer transactions and profile tables (orders, customers, transactions).

Tools: SQL (joins, aggregations), Power BI (visualization, dashboarding).

Approach: Performed RFM-style segmentation using SQL to compute recency, frequency, and monetary value metrics. Built an interactive Power BI dashboard to explore segments and track KPIs.

Outcome: Identified top 10% customers generating ~45% of revenue and created a dashboard with filters to guide targeted retention campaigns.

Customer Segmentation — Notes

SQL snippets used: GROUP BY customer_id, SUM(revenue), COUNT(order_id), joins between orders and customers for enrichment. Sample KPIs included ARPU, churn rate, and cohort retention.

Recommendations: Run A/B tests on targeted offers to the high-value cohort and monitor uplift via the dashboard.
